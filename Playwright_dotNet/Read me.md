**Prerequisites**

	**IDE (VS CODE):** We can use other IDE as well but in this I used VS Code

		Create a base project in your preferred location using nunit framework (also supports mstest and xunit but nunit is a good option to go for)
		we can use terminal to create project by - "dotnet new nunit" we can also pass -n to create in a separate folder
		or we can create one using visual studio
		

	**Playwright:**
		
		we can add playwright using nuget packages with name "Microsoft.Playwright.NUnit", we can use others like MStest as well.
		or we can add them using terminal vy following this - https://playwright.dev/dotnet/docs/intro
