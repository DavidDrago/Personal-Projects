**Prerequisites**



	**Java:** To check if java Installed -> Java -version

	

&nbsp;	**IDE:** Eclipse, IntelliJ, etc...

&nbsp;	

&nbsp;	**Maven:** For Dependencies, To check if installed -> mvn -v

&nbsp;		Maven is already available with IDE, but if we want to run the code in command line we need to install it through apache maven.



&nbsp;	**TestNG:** Testing Framework



	**RestAssured:** Can be added to the Maven POM.xml as a dependency found in Maven Repository


	**Json-Server:** Used for dummy data for API testing.
		Need Node.Js to work. To check if available use "node -v" or "node --version"
		Also check for npm by "npm -v"
		install Json-Server by "npm install json-server" and to install globally use "npm install -g json-server"
		create a db.json in respective folder and run "npx json-server db.json"
		Follow through this link for more info - https://www.npmjs.com/package/json-server?activeTab=readme
